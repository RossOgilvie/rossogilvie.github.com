32.
Das Mathematische Pendel.
Gegeben sei die Schwingungsgleichung des Mathematischen Pendels mit Reibung

ẍ = −ẋ − g ⋅sin⁡(x). 		(32.1)

Die gesuchte Funktion t↦x(t) ∈ ℝ ist der Winkel von Senkrechte. Die physikalische Bedeutung der Konstanten g ist die Fallbeschleunigung; die andere Konstanten (Masse, Reibung) haben wir als 1 gesetzt. Die ODE zweiter Ordnung kann in ein erste Ordnung System transformiert werden:

d 𝑑𝑡y = d 𝑑𝑡 ( x ẋ ) = ( 0 1 0 −1 ) ( x ẋ )+ ( 0 −g sin⁡ x ) = f(y) 		(32.2)
