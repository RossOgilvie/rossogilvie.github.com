3.
Lösungen von Differentialgleichungen.
(a)
Sei m ∈ ℝ mit m > 0. Die Konstant g ist 9,8ms−2, lassen Sie aber als g. Finden Sie eine Lösung q für die Differentialgleichung mq″ = −𝑔𝑚 und mit q(0) = 10 und q′(0) = 0. (2 Punkte)
(b)
Zeigen Sie: Für α,A,B ∈ ℝ beliebig ist y(t) := Acos⁡(𝛼𝑡) + Bsin⁡(𝛼𝑡) eine Lösung der Differentialgleichung y″ + α2y = 0. (1 Punkt)
(c)
Sei f(t,u,u′) := 4u′− 3u − 3t + 4. Zeigen Sie, dass u(t) = Aet + Be3t − t mit A,B ∈ ℝ eine Lösung der Differentialgleichung u″ = f(t,u,u′) ist. Bestimmen Sie A und B, sodass u(0) = u′(0) = 0 gilt. (4 Punkte)
(d)
Finden Sie eine Differentialgleichung, für die die Funktion y(t) = Aet + Be−t + C für alle A,B,C ∈ ℝ eine Lösung ist. (4 Punkte)

[Tipp: Man überlege zunächst, welche Ordnung die Differentialgleichung haben sollte. Dann berechne man y′,y″,… und überlege, wie man die Parameter A,B,C loswird.]
